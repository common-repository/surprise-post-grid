<h4>How To Use</h4>
<p>We pre made shortcode. So you can easily use anywhere your post by shortcode. Here is some example</p>

<pre>
 [surprise_post_grid style="style-1"] style available  style-1, style-2, style-3, style-4, style-5, style-6
</pre>

<p>You can change column by column, See the example below</p>
<pre>
 [surprise_post_grid style="style-1" column="3"] 
 </pre>
<p>More example </p>
<pre>
 [surprise_post_grid style="style-1" column="3" category="your-category-id" tag="your-tag-id" pagination="yes" orderby="date" order="DESC"] 
 </pre>
